# Features

- 트위치 생방송, 트위치 다시보기, popout 채팅창에서 작동
- 버튼 클릭 시 디시콘 목록 보기
- 채팅창에 `~` 입력 시 디시콘 목록 검색 가능
- 디시콘 목록에서 `↑`, `↓`으로 탐색, `→`으로 입력. !문제점: 위 방향키를 누르면 입력의 처음으로 감. 그 상태로 복사가 되서 작동하지 않음.
- 디시콘 목록에서 클릭시 입력창에 붙여넣기 또는 클립보드에 복사
- 채팅창에서 디시콘 클릭 시 입력창에 붙여넣기 또는 클립보드에 복사
